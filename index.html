The basics of the Web and HTML - lesson notes The World Wide Web

The web consists of c. 30 billion pages today and was invented in the 1990s. It largely consists of pages that are generated by HTML code (HyperText Markup Language), and pages link to each other through Hyper Links.

The major components of the web

The web can largely be broken down into 3 major component parts; 1) The user, computer and browser (e.g. Chrome, Internet Explorer, Firefox, Safari) 2) The internet, being a collection of hyperlinked webpages largely constructed using HTML 3) Servers that contain and store large repositories of data.

A user accesses the internet through a web browser and information is relayed to and from servers via a protocol; HTTP. This protocol determines how information is relayed across the internet.

HTML basics

HTML is made up of; 1) text content, 2) markup, 3) references to other documents (e.g. videos, images), 4) links to other pages.

Markup determines how text will ultimately look on a webpage, and it comes in the form of tags. Tags generally consist of three parts; 1) opening tag, 2) content, 3) closing tag.

These three parts are together referred to as an ‘element’.

Examples of commonly used HTML tags are:

1) <b>content</b>
This tag makes the text bold.

2) <em>content</em>
This tag makes the text italicised.

Note that if a closing tag is required to determine how much content the tags apply to. If it is missing, all of the content following on from the opening tag will be e.g. bold or italicised.

HTML attributes

HTML tags can have attributes that determine the behaviour of the tag. They usually appear in the following form:

<TAG ATTR=”value”>content</TAG>

An example of a tag that uses attributes is the anchor tag: <a>. The anchor tag applies a hyperlink to the text:

<a href="www.reddit.com">derp</a>

In this case, the 'derp' text will appear in blue and underlined, indicating that the text is hyperlinked, i.e. by clicking on the text, the user will be taken to www.reddit.com

Another tag that uses attributes is the image tag. It comes in the following form:

<img src=”url” alt=”text”>

The 'src' attribute indicates the source of the image, i.e. website address, and the 'alt' attribute indicates the text that should appear in case the image cannot be accessed.

Whitespace

All spaces in text, whitespaces, are largely ignored in HTML. E.g. consider the following text:

This is the        text content
in a      sentence.

This text would appear as a single sentence without a linebreak and with only one space between each word.

In order to enter a linebreak, the <br> tag is used (break). This is a so called 'void' tag, i.e. is does not require a closing tag. The <p> tag can also be used, i.e. the paragraph tag, although this tag requires a closing tag.

Inline vs. block tags

Tags can be broken down into two types;

1) inline – used within a sentence, e.g. <br> <b> <em> <img> <span>,

2) block – frames the text within an invisible box, e.g. <p> <div>.

Inline elements do not cause text to wrap down into the next line while block elements do.

Span/div elements

Both of these elements are so called 'container' elements with <span> being inline and <div> being block. Both of these elements have a 'class' attribute where CSS styles and Javascript code can be used to alter the visual style of the content:

<span class=”foo”>text</span>

<div class=”bar”>text</div>

HTML documents

All HTML documents require several component parts in order to fulfil the requirements of HTML markup. They are as follows:

<!DOCTYPE HTML>

<html>

<head>

<title>Title!</title>

</head>

<body>
<b>content</b>
</body>

</html>

The first tag, <!DOCTYPE HTML>, indicates the type of page it is. The opening and closing <html></html> tags surround the entire html document. The <head></head> tags contain metadata, CSS/Javascript styling information, and the <title> tag, indicating the overall title of the document. The<body> tags contain the contents of the document.
